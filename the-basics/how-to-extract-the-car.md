# How to extract the car

In order to get the car in blender, we first need to import the car in 3DSimed

Chose the import option of your choice depending on the game which car you extract



<figure><img src="../.gitbook/assets/image (111).png" alt=""><figcaption><p>The RB6 from codemaster</p></figcaption></figure>

Export is as collada and make sure you tick copy texture option.

<figure><img src="../.gitbook/assets/image (219).png" alt=""><figcaption></figcaption></figure>

Once it's done, you can start blender, and load the .cad file you exported

<figure><img src="../.gitbook/assets/image (84).png" alt=""><figcaption><p>*</p></figcaption></figure>

You can now start working on the car to convert it to Assetto Corsa
